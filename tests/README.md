# Тестирование Calorie Counter Bot

## Установка зависимостей

Перед запуском тестов убедитесь, что установлены все необходимые зависимости:

```bash
npm install
```

## Запуск тестов

### Запуск всех тестов
```bash
npm test
```

### Запуск тестов в режиме наблюдения
```bash
npm run test:watch
```

### Запуск конкретного теста
```bash
npx jest tests/mainFlow.test.js
npx jest tests/limits.test.js
npx jest tests/correction.test.js
npx jest tests/errorHandling.test.js
```

## Конфигурация тестов

### Jest Configuration (jest.config.js)
- Тестовая среда: Node.js
- Паттерн поиска: `**/tests/**/*.test.js`
- Таймаут: 30 секунд
- Setup файл: `tests/setup.js`

### Переменные окружения (.env.test)
- Тестовые токены и ключи API
- Отдельная тестовая база данных
- Минимальный уровень логирования

## Моки и заглушки

Тесты используют Jest моки для:
- OpenAI API (`src/services/openai`)
- Сервисы пользователей и запросов
- База данных queries
- Логирование

Это позволяет тестировать логику без зависимости от внешних сервисов.

## Запуск в CI/CD

Для автоматического тестирования в CI/CD добавьте:

```yaml
# .github/workflows/test.yml
name: Tests
on: [push, pull_request]
jobs:
  test:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v2
      - uses: actions/setup-node@v2
        with:
          node-version: '18'
      - run: npm install
      - run: npm test
```

## Отладка тестов

Для отладки конкретного теста:

```bash
# Запуск с подробным выводом
npx jest tests/mainFlow.test.js --verbose

# Запуск только одного теста
npx jest tests/mainFlow.test.js -t "should process photo without weight"

# Запуск с покрытием кода
npx jest --coverage
```
