# 🚀 Быстрый старт: Профиль пользователя

## Установка за 3 шага

### 1️⃣ Выполнить миграцию
```bash
npm run migrate
```

Должны увидеть:
```
✅ Все миграции выполнены успешно
   ✓ init.sql
   ✓ add_user_profiles.sql
```

### 2️⃣ Перезапустить бота
```bash
npm restart
```

### 3️⃣ Проверить работу
Отправьте боту в Telegram:
```
/profile
```

## ✅ Готово!

Теперь пользователи могут:
- ✅ Использовать бота БЕЗ профиля (как раньше)
- ✅ Создать профиль для расчета нормы калорий
- ✅ Видеть прогресс по цели при анализе фото

## 📖 Документация

- `PROFILE_FEATURE.md` - полная документация функции

## 🎯 Основные команды

```
/profile - управление профилем (опционально)
/status - статус + цель калорий (если профиль есть)
/start - приветствие
```

## 💡 Важно

**Профиль НЕ обязателен!** Бот полностью работает без профиля.

Профиль нужен только для:
- Расчета персональной нормы калорий
- Отслеживания прогресса по цели
